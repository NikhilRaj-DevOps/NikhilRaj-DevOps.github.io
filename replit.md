# Nikhil Raj — Portfolio

A personal portfolio website for Nikhil Raj, Senior DevOps / SRE Engineer.

## Stack

- **Pure static site**: HTML, CSS, JavaScript (no build step)
- **Fonts**: JetBrains Mono + Inter (Google Fonts CDN)
- **Theme**: Dark tech — dark background with neon blue/cyan accents

## Running the site

The site is served with Python's built-in HTTP server:

```bash
python3 -m http.server 5000
```

Open `http://localhost:5000` (or use the Replit preview pane).

## Project structure

```
index.html   — Main portfolio page (all sections)
styles.css   — All styling (dark theme, responsive)
script.js    — Typed text effect, scroll animations, mobile nav
README.md    — Repo description
```

## Sections

1. **Hero** — Name, animated role title, stats bar
2. **About** — Bio + terminal animation card
3. **Skills** — Categorised tech stack cards
4. **Experience** — Timeline: Aziro, UST, Cognizant with projects
5. **Awards** — USTAR Super Star, Innovation Award, AZ-104 cert
6. **Contact** — GitHub + LinkedIn links

## User preferences

- Dark tech visual style (dark BG, neon blue/cyan accents)
- JetBrains Mono for code/terminal elements, Inter for body text
